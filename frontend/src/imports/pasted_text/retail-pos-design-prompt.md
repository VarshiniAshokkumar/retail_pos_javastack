# Figma AI Prompt — Retail Grocery POS System for Small Shops

Design a modern, clean, easy-to-use POS (Point of Sale) software UI for a small grocery shop / mini supermarket in India.
The software should work like a simplified Dmart billing system but designed for small local shops that currently use paper and pen for billing.

The design must focus on:

* Simple workflow
* Fast billing
* Easy product search
* Large readable buttons
* Minimal training required
* Responsive desktop-first layout
* Touch-friendly interface
* Professional supermarket style
* Modern dashboard with soft colors
* Tamil Nadu local grocery shop feel with simple usability
